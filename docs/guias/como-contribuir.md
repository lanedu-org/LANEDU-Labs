# Cómo contribuir a LANEDU Labs

Gracias por querer aportar.

Este repositorio está pensado para aprender,
no para demostrar que sabes más.

---

## Qué tipo de aportes son bienvenidos

- Mejorar explicaciones
- Simplificar lenguaje
- Corregir errores
- Agregar ejemplos

---

## Buenas prácticas

- Un cambio por Pull Request
- Cambios pequeños
- Mensajes claros
- Respeto en los comentarios

---

## Recuerda

> Aquí se evalúa el proceso, no la perfección.
