# Python

## ¿Qué es Python?

Python es un lenguaje de programación de alto nivel, enfocado en la legibilidad y simplicidad.

Se utiliza en:
- Desarrollo web
- Automatización
- Ciencia de datos
- Inteligencia artificial

---

## ¿Para qué se usa?

Python se usa cuando:
- Se busca rapidez de desarrollo
- Se quiere código fácil de leer
- Se trabaja con muchas librerías
