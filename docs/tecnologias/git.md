# Git

## ¿Qué es Git?

Git es un sistema de control de versiones.
Permite registrar cambios y volver atrás si algo falla.

---

## ¿Para qué se usa?

- Controlar cambios en proyectos
- Trabajar en equipo
- Mantener historial

---

## Conceptos clave

- Repositorio
- Commit
- Rama
- Merge

---

## Errores comunes

- No hacer commits frecuentes
- No escribir mensajes claros
- Trabajar siempre en main
