# GitHub Basics – Fundamentos para LANEDU Labs

GitHub es una plataforma que permite **trabajar en equipo sobre proyectos**, manteniendo un historial claro de cambios.

En LANEDU Labs usamos GitHub para:
- Colaborar sin pisarnos
- Proponer mejoras de forma ordenada
- Aprender cómo funcionan los equipos reales

---

## ¿Qué es GitHub?

GitHub es un servicio que aloja repositorios y utiliza Git para llevar el control de versiones.

No es solo para programar.
También sirve para:
- Documentación
- Aprendizaje
- Proyectos colaborativos

---

## Conceptos que usarás aquí

En este laboratorio trabajarás con estos conceptos:

- **Repositorio**: el proyecto completo
- **Issue**: una tarea, problema o mejora
- **Rama (branch)**: una línea de trabajo separada
- **Commit**: un registro de cambios
- **Pull Request**: una solicitud para integrar cambios

No necesitas dominarlos de inmediato.
Los aprenderás usándolos.

---

## Idea clave

> En GitHub, lo más importante no es el código.  
> Es la **comunicación y el proceso**.
