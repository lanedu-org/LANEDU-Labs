# Glosario – LANEDU Labs

Este glosario explica conceptos usados en GitHub y en este laboratorio.

---

## Repositorio
Proyecto que contiene archivos, documentación e historial de cambios.

---

## Issue
Registro de una tarea, problema o mejora pendiente.

---

## Fork
Copia de un repositorio en tu propia cuenta.

---

## Rama (Branch)
Línea de trabajo separada del proyecto principal.

---

## Commit
Registro de cambios realizados en un proyecto.

---

## Pull Request
Solicitud para integrar cambios a un repositorio.

---

## Main
Rama principal del proyecto.

---

Este glosario está vivo.
Si puedes mejorar una definición, hazlo.
