# Preguntas frecuentes (FAQ)

Esta sección responde las dudas más comunes al usar LANEDU Labs.

Si tu duda no está aquí, revisa primero la documentación del Lab.

---

## ¿Necesito subir archivos a la plataforma?

No.

LANEDU Labs **no recibe archivos** directamente.

Toda la entrega se realiza mediante un Pull Request en GitHub.

---

## ¿Dónde entrego mi solución?

Tu entrega es un **Pull Request** contra el repositorio base del Lab.

Si no existe un Pull Request válido, el Lab no puede completarse.

---

## ¿Puedo hacer merge del Pull Request?

No.

Los Pull Requests **no se mergean**.

Sirven únicamente como evidencia para validación.

---

## ¿Por qué mi Lab sigue en “Pendiente”?

Las causas más comunes son:

- No existe un Pull Request
- El Pull Request tiene un título incorrecto
- El Pull Request apunta a la rama equivocada

Revisa las reglas del Lab.

---

## ¿Por qué aparece “PR detectado” pero no “Completado”?

Esto significa que el Pull Request existe, pero **no cumple todas las reglas**.

Revisa:
- Título del PR
- Archivos requeridos
- Archivos prohibidos

Corrige el mismo Pull Request.

---

## ¿Puedo abrir más de un Pull Request?

No.

Cada Lab se entrega con **un solo Pull Request**.

Si necesitas corregir algo, actualiza ese Pull Request.

---

## ¿Puedo usar GitHub Codespaces?

Sí.

Puedes usar GitHub Codespaces o tu entorno local.

La plataforma no depende del entorno que utilices.

---

## ¿Puedo modificar el repositorio base del Lab?

No.

El repositorio base es inmutable.

Todo el trabajo ocurre en tu fork.

---

## ¿Qué pasa si borro archivos por error?

Si el Lab prohíbe borrar archivos, la validación fallará.

Revisa siempre las reglas antes de entregar.

---

## ¿Los Labs tienen solución única?

No.

Mientras cumplas el objetivo y las reglas, cualquier solución válida es aceptada.

---

## ¿Puedo rehacer un Lab ya completado?

Sí.

Puedes repetir un Lab cuantas veces quieras, pero solo se valida una vez por entrega.

---

## ¿Necesito conocimientos previos?

Depende del Lab y la Ruta.

Las Rutas comienzan en nivel básico y avanzan progresivamente.

---

## En resumen

LANEDU Labs:
- Valida por Pull Requests
- No hace merges
- No recibe archivos
- Funciona 100% con GitHub
