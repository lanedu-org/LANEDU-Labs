# LANEDU Labs

LANEDU Labs es una plataforma de aprendizaje **100% práctica**, basada en desafíos reales y validación automática mediante GitHub.

Aquí no hay cursos tradicionales ni teoría extensa.  
Aprendes **haciendo**, como en un entorno de trabajo real.

---

## ¿Qué es LANEDU Labs?

LANEDU Labs es un sistema de Labs prácticos organizados por **Rutas**.

Cada Lab plantea un problema concreto, con contexto realista y reglas claras.  
La solución se entrega mediante un **Pull Request**, que es validado automáticamente.

No hay formularios.  
No hay archivos subidos manualmente.  
Todo ocurre en GitHub.

---

## ¿Cómo funciona?

El flujo es siempre el mismo:

1. Eliges un Lab
2. Forkeas el repositorio
3. Resuelves el desafío
4. Abres un Pull Request
5. La plataforma valida automáticamente

Así de simple.

---

## ¿Qué es un Lab?

Un Lab es un desafío práctico que simula una tarea real de trabajo.

Cada Lab:
- Vive en un repositorio propio
- Tiene una historia y un objetivo claro
- Define reglas de validación
- Se entrega mediante un Pull Request

No hay una única solución correcta.  
Mientras cumplas el objetivo y las reglas, tu entrega es válida.

---

## ¿Qué es una Ruta?

Una Ruta es una secuencia de Labs relacionados por una misma tecnología o área.

Ejemplos de Rutas:
- Linux
- Git y GitHub
- Bases de datos
- Automatización
- DevOps

Las Rutas avanzan de nivel básico a avanzado.

---

## Validación automática

LANEDU Labs valida las entregas automáticamente usando:

- Convenciones de Pull Requests
- Reglas definidas en cada Lab
- Estado real del repositorio en GitHub

No hay revisión manual ni correcciones subjetivas.

---

## Para quién es LANEDU Labs

LANEDU Labs es para personas que quieren:

- Aprender haciendo
- Practicar con problemas reales
- Mejorar habilidades técnicas
- Trabajar con GitHub de forma profesional

No necesitas saberlo todo antes de empezar.  
Las Rutas comienzan desde lo básico.
