# ¿Qué es LANEDU Labs?

LANEDU Labs es una plataforma de aprendizaje **100% práctica**, diseñada para aprender haciendo tareas reales, no estudiando teoría.

Aquí no hay clases grabadas ni ejercicios artificiales.  
Cada desafío simula una situación real de trabajo, como si ya formaras parte de un equipo técnico.

---

## Aprender trabajando

En LANEDU Labs aprendes resolviendo problemas reales:

- Un jefe o lead te pide algo concreto
- Hay un objetivo claro
- Existen restricciones y reglas
- Debes entregar una solución funcional

Exactamente como en un entorno profesional.

---

## Labs, no cursos

La unidad principal de aprendizaje es el **Lab**.

Un Lab es:
- Un problema real
- Un repositorio de GitHub
- Una entrega validada por Pull Request

No sigues lecciones.
No completas formularios.
No subes archivos a plataformas externas.

Todo ocurre en GitHub.

---

## Validación con evidencia real

LANEDU Labs valida tu progreso automáticamente usando GitHub:

- Fork del repositorio del Lab
- Commits con tu solución
- Pull Request contra el repo base

Si el Pull Request cumple las reglas del Lab, el desafío se marca como completado.

El Pull Request es la evidencia.
No hay revisión manual.

---

## Rutas de aprendizaje

Los Labs se organizan en **Rutas**.

Una Ruta agrupa Labs del mismo dominio, ordenados de **fácil a difícil**.

Ejemplos:
- Ruta Linux
- Ruta Git & GitHub
- Ruta Bases de Datos
- Ruta Docker

Completar una Ruta demuestra dominio práctico de ese tema.

---

## ¿Para quién es LANEDU Labs?

LANEDU Labs es para personas que:

- Quieren aprender habilidades reales
- Prefieren practicar antes que memorizar
- Quieren construir evidencia pública de su trabajo
- Buscan experiencia aplicable al mundo laboral

No importa tu nivel inicial.
Importa tu capacidad de resolver problemas.

---

## En resumen

LANEDU Labs no enseña teoría.  
Enseña a **trabajar**.

Aprendes haciendo.
Entregas con Pull Requests.
Demuestras con evidencia real.
